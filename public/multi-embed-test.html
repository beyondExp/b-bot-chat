Technical Details:

Each embed has a unique embedId parameter which ensures:

  • Isolated localStorage: Each embed uses keys like embed-chat-history-embed-1
  • Independent sessions: Thread IDs and chat history are stored separately
  • Separate branching: Message editing and branching work independently per embed
  • Different agents: Each embed can use different AI agents or configurations

localStorage Keys Used:

  • embed-chat-history-embed-1 - Chat history for Embed #1
  • embed-current-thread-embed-1 - Current thread ID for Embed #1
  • embed-chat-history-embed-2 - Chat history for Embed #2
  • embed-current-thread-embed-2 - Current thread ID for Embed #2
  • ...and so on for each embed
