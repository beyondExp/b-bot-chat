Embedded Chat Example

This page demonstrates how to embed the B-Bot chat interface in any website.

How to Embed

To embed this chat interface in your own website, add the following HTML code:

<iframe 
  src="https://your-domain.com/embed?agent=bbot&embedId=unique-id" 
  style="width: 100%; height: 600px; border: none;"
  title="B-Bot Chat"
  allow="microphone"
></iframe>

URL Parameters

  • agent - The AI agent to use (e.g., bbot, open_deep_research, retrieval_graph)
  • embedId - Unique identifier for this embed instance (required for multiple embeds)
  • user_id - Optional user identifier for personalized chat history

Multiple Embeds on Same Page

If you need multiple chat embeds on the same page, each must have a unique embedId parameter to ensure independent chat histories:

<!-- First embed -->
<iframe src="https://your-domain.com/embed?agent=bbot&embedId=embed-1"></iframe>

<!-- Second embed -->
<iframe src="https://your-domain.com/embed?agent=bbot&embedId=embed-2"></iframe>

Without unique embedIds, all embeds would share the same conversation history.

Example URLs

  • ?agent=bbot&embedId=support-chat - Default B-Bot for customer support
  • ?agent=open_deep_research&embedId=research-assistant - Research agent
  • ?agent=retrieval_graph&embedId=doc-search&user_id=user123 - Document search with user ID
