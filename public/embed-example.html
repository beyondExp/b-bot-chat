Embedded Chat Example

This page demonstrates how to embed the B-Bot chat interface in any website.

How to Embed

To embed this chat interface in your own website, add the following HTML code:

<iframe 
  src="https://your-domain.com/embed?agent=b-bot" 
  style="width: 100%; height: 600px; border: none;"
  title="B-Bot Chat"
  allow="microphone"
></iframe>

You can specify different agents by changing the agent parameter in the URL:

  • ?agent=b-bot - Default B-Bot
  • ?agent=agent-id - Replace with any agent ID from your platform
